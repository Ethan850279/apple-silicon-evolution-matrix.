 Apple Silicon Architecture Matrix & Export Simulator story
License: MIT Platform: Live
This repository contains the source code for historyofapplesilicon.com, an interactive web application that explores ten generations of Apple custom system-on-chip microarchitectures. The site is built with responsive HTML5, modern CSS3 variables, and vanilla JavaScript.


  	  


🛠️ Software Overview check
The project includes an integrated Video Processing Telemetry Simulator. This tool estimates rendering throughput speeds using scale coefficients derived from practical computing metrics.
📊 Computational Model
The simulation engine models processing performance using baseline multi-threaded metrics, based on typical commercial video encoding workloads:
• M1 Base Layer: 15 seconds per video minute
• M2 Max variant: 9 seconds per video minute
• M3 Ultra node: 5 seconds per video minute
• M4 Pro acceleration: 2 seconds per video minute
• M5 Pro (2nm Node Profile): 1.2 seconds per video minute
These values are intended as indicative benchmarks for comparative purposes within the simulator.


  	  


📁 Repository Structure
• `index.html` – Defines the main layout, viewports, and the JavaScript logic that powers the simulation.
• `style.css` – Provides the visual presentation layer, including Apple-inspired dark styling, typography, and a responsive grid system.
